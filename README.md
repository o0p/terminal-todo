# terminal-todo
Terminal based to-do program with the focus on having smallest footprint as possible
    <h2>This prototype will work in your browser's console</h2>
    <h5>List of available commands:</h5>
    <ul>
        <li>Add item: n</li>
        <li>Delete item: d</li>
        <li>List all items: l</li>
        <li>Quit: q</li>
    </ul>
